# ASCIIChess

This project was completed as a final project for CS 410P: Advanced C++, taught by Steven Libby at Portland State University during the Winter 2019 term.

## How to run

1. Download the files
2. Compile: ```g++ *.cpp```
3. Run: ```./a.out```
